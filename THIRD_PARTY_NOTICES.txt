Translation provider
--------------------

Audio Mangler can translate to English locally with Whisper.
For non-English target languages, Audio Mangler can call OpenAI if OPENAI_API_KEY is set.

Practical notes:

- OpenAI is not bundled with this repository.
- OpenAI API usage is subject to your own account, API key, pricing, and service terms.
- Generated translations are output from a third-party service path and should be reviewed before reuse or publication.
