optional workflows that rely on third-party media, tools, or services.

Optional tooling and services
-----------------------------

Media Manglers can use third-party tools and services that are not owned by
this repository.

Examples include:

- FFmpeg for media processing
- Whisper for local transcription and some local translation paths
- yt-dlp for supported remote media downloads and metadata retrieval
- Argos Translate for free local translation to non-English targets
- OpenAI for optional cloud translation when you configure an API key

Practical notes:

- These tools and services are separate from the Apache License 2.0 that covers this
  repository's code.
- OpenAI is optional and not bundled with the repo.
- OpenAI API usage depends on your own account, key, pricing, and service
  terms.
- Generated transcripts and translations should be reviewed before you rely on
  them for publication or downstream work.

Remote site content
-------------------

When you point either app at YouTube or another supported remote source, the
media, metadata, and public comments remain subject to that source platform and
the rights status of the underlying content.

General rule
------------

Do not assume that referenced media, metadata, or third-party services become
Apache 2.0 licensed just because this repository is Apache 2.0 licensed.
If you redistribute any referenced media or exported content, follow the
upstream terms for that specific source.
